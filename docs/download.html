Download StatConverter

Disclaimer: the application files are generated as platform specific binaries for your convenience only. However (at least currently), they are not digitally signed for authenticity. The operating system will ask for all sorts of permissions to install / run the application and will warn it is unknown and unsafe.

 

If you do not trust these binaries, the application can be started from its source files, or build your own binaries (see the left menu for instructions).

 

Windows 64bit

Executable installation file (needs admininstrative privileges):

StatConverter_1.2.0_setup_x86_64.exe

 

If you do not have administrative privileges to install, or simply don't want to install the application, it can be downloaded and directly run as an executable file:

StatConverter_1.2.0_x86_64.zip

 

Windows 32bit

Executable installation file (needs admininstrative privileges):

StatConverter_1.2.0_setup_x86_32.exe

 

Zip file (no need to install):

StatConverter_1.2.0_x86_32.zip

 

MacOS 64 bit Apple Silicon (ARM)

Disk image file (needs admininstrative privileges):

StatConverter_1.2.0_arm64.dmg

 

MacOS 64 bit Intel x86 Big Sur and higher

Disk image file (needs admininstrative privileges):

StatConverter_1.1.9_Intel_x86_64_Big_Sur.dmg

 

MacOS 64 bit Intel x86 High Sierra and higher (up to R 4.2.3)

Disk image file (needs admininstrative privileges):

StatConverter_1.1.9_Intel_x86_64_High_Sierra.dmg

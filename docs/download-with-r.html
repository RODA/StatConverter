Download StatConverter (including R, embedded)

Disclaimer: the application files are generated as platform specific binaries for your convenience only. At least currently, they are not digitally signed however. The operating system will ask for all sorts of permissions to install / run the application and will warn the application is unknown and unsafe.

 

If you do not trust these binaries, the application can be started its source files, or compile your own binaries (see the left menu for instructions).

 

On Windows, R can be made portable and packed into the converter.

 

For Windows 64bit

Setup file (needs admininstrative privileges):

StatConverter_setup_R_embedded_1.0.1.exe

 

If you do not have administrative privileges to install, or don't want to install the application, it can be run without installation:

StatConverter_R_embedded_1.0.1.zip

 

For Windows 32bit

Setup file (needs admininstrative privileges):

StatConverter_setup_32bit_R_embedded_1.0.1.exe (link soon to be added)

 

Zip file (no need to install):

StatConverter_32bit_R_embedded_1.0.1.zip (link soon to be added)

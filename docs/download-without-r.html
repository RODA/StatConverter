Download StatConverter

Disclaimer: the application files are generated as platform specific binaries for your convenience, but (at least currently) are not digitally signed. The operating system will ask for all sorts of permissions to install / run the application and will warn the application is unknown and unsafe.

 

If you do not trust these binaries, you can run the application from sources or compile your own binaries (see the left menu for instructions).

 

If you already have your own R, or want to install R separately (see the CRAN download page), the files below are smaller because they do not contain R embedded. When R is successfully installed on the local computer, StatConverter can directly communicate with it.

 

For Windows 64bit

R must be on the system PATH, see further instructions on the left menu, on installing R for Windows.

 

Setup file (needs admininstrative privileges):

StatConverter_setup_1.0.1.exe

 

Zip file (no need to install):

StatConverter_1.0.1.zip

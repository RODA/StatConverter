Download StatConverter (R installed separately, not embedded)

Disclaimer: the application files are generated as platform specific binaries for your convenience only. However (at least currently), they are not digitally signed for authenticity. The operating system will ask for all sorts of permissions to install / run the application and will warn it is unknown and unsafe.

 

If you do not trust these binaries, the application can be started its source files, or compile your own binaries (see the left menu for instructions).

 

If you already have your own R, or want to install R separately (see the CRAN download page), the files below are smaller because they do not contain R embedded. When R is successfully installed on the local computer, StatConverter can directly communicate with it if found on the system path (by default on MacOS and Linux, manual operation on Windows).

 

The MacOS binary is built for the Intel architecture, and it assumes a default location of the R executable in: /usr/local/bin/R

 

For Windows 64bit

R must be on the system PATH, see further instructions on the left menu, on installing R for Windows.
